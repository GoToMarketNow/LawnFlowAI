3) AGENT LOGIC UPDATES
================================================================================

ServiceSelectionAgent:
- If mulch requested:
  - ask if delivery only or delivery + installation
- If firewood requested:
  - ask one-time vs recurring vs seasonal

PricingAgent:
- Use stored MulchProfile / FirewoodProfile if exists
- Otherwise estimate and flag as first-install confidence adjustment

SchedulingAgent:
- Enforce lead-time
- Prevent same-day scheduling
- Bundle delivery + install where applicable

QuoteAgent:
- Separate material vs labor in explanation
- Highlight savings for recurring services
